## 📋 What is std-semver?

std-semver is a tool that helps you check and understand software version numbers. It follows a system called "semantic versioning" or semver. This system uses numbers to tell if a software update adds new features, fixes bugs, or changes things in a way that might break other software.

Since this tool is a simple port of a known version manager, it brings its functions to your Windows computer without any complicated steps. You can easily see if software versions are up-to-date or if you need to update to a safe, newer version.

---

## 🎯 Who should use this?

You do not need to be a programmer to use std-semver. If you need to work with software versions for any reason—whether you install programs, maintain apps, or check software updates—this tool can help.

It works well for:

- Users who want to keep software organized by version.
- People who want to verify if they have the latest software update.
- Anyone curious about how software version numbers change.

## ⚙️ How to Use std-semver

After launching the program, you will see a simple window. You can start using std-semver by:

1. **Inputting Versions:** Type in two version numbers you want to compare. For example, "1.2.0" and "1.2.1."
2. **Check Version Difference:** The program will explain if the update is major, minor, or a patch.
3. **Validate Versions:** You can make sure a version number follows the correct format.
4. **Sort Versions:** Input multiple versions, and std-semver will sort them from oldest to newest.

This makes it easier to know which software update is important.
